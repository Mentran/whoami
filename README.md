# 我是谁

一个宝可梦粉丝向的「Who's That Pokemon?」复古掌机猜影子小游戏。

## 产品目标

打开页面后，用户直接进入游戏：看到一只宝可梦黑影，输入答案，答对后揭晓真身并计分。第一版专注于节奏、视觉氛围和稳定可玩，不做后端、登录或语音识别。

## MVP 范围

- 复古红色掌机界面
- 第一世代宝可梦题库
- 黑影猜谜
- 中文名 / 英文名 / 常见别名判定
- 答对、答错、跳过、下一题
- HIT / TOT 计分
- 本地最高分
- 桌面和移动端适配

## 技术方案

- Vite
- React
- TypeScript
- 原生 CSS
- 本地题库 + PokeAPI official artwork 图片 URL
- `localStorage` 保存最高分

## 里程碑

### M1：可玩闭环

- 初始化前端项目结构
- 实现掌机外壳和游戏屏幕
- 内置一批宝可梦题目
- 完成输入、判题、揭晓、计分

### M2：复古质感

- CRT 扫描线
- 答错震动
- 答对闪屏
- Reveal 动画
- 按钮触感

### M3：内容扩展

- 扩展到第一世代 151 只
- 增加难度和连续挑战
- 支持更多别名

### M4：传播增强

- 分享结果图
- 每日挑战
- 自定义题库草案
